# SmartSummarizer AI

An NLP-powered text summarization tool that converts lengthy articles and documents into concise, easy-to-read key insights.

## Features

* Automatic Text Summarization
* Key Insight Extraction
* Reading Time Analysis
* Compression Statistics
* Download Summary
* Dark & Light Mode Support

## Technologies Used

* Python
* Flask
* NLTK
* Sumy
* HTML, CSS, JavaScript

## How It Works

The application uses extractive summarization techniques to identify and extract the most important sentences from a document, reducing reading time while preserving key information.

## Analytics Dashboard

* Original Word Count
* Summary Word Count
* Compression Percentage
* Reading Time Saved
* Quality Score

## Future Improvements

* PDF Summarization
* Multi-language Support
* Abstractive Summarization
* Document Upload Feature
